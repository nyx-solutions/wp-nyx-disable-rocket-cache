Disables WP Rocket’s page cache while preserving other optimization features.

== Description ==
Disables WP Rocket’s page cache while preserving other optimization features. In order to enable the functionality, use the constant NYX_DISABLE_ROCKET_CACHE with true

== Installation ==
In order to enable the functionality, use the constant NYX_DISABLE_ROCKET_CACHE with true.
